# redux

This repository is a collection of react apps created with redux.
Each branch is a different app.

## How to run different apps
- `git clone` the repository
- `git checkout <branch-name>`
- `npm install`
- `npm run start`

## What is being used?
- NPM
- Node
- React
- Redux
- Babel
- Webpack
